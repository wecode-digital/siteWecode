

<h1 align="center">Site Wecode</h1>

## Descrição
<p>Projeto desenvolvido em React </p>



## Pré-requisitos
- Ter o *Git* ou *GitHub Desktop* instalado para clonagem do repositório
- Ter o *NodeJs* instalado para rodar o servidor do projeto
<p>Após clonar o repositório, rodar os seguintes comandos na pasta do projeto:</p>
<p><i>yarn add gulp gulp-sass node-sass gulp-sourcemaps</i> - instalar dependências</p>
<p><i>npm install</i> - instalar dependencias</p>
<p><i>npm start</i> - rodar o servidor NodeJs do projeto</p>




### Tecnologias utilizadas

- ReactJs
- NodeJs
- React Router Dom
- Sass
